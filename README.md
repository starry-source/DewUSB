## 简介
一个简约美观的U盘助手

## 正常使用

- 在右侧 Releases 中下载最新版本的压缩包（`.rar` 文件）。
- 解压到你喜欢的位置，双击 `DewUSB.exe` 即可运行。

## 代码部署

克隆存储卡，双击 `DewUSB.sln` 打开 Visual Studio，配置好依赖即可。

## 开发环境

Windows 11 24h2\
.NET Framework 4.7.2
